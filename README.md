# 浙江工商大学 · 首届 AI 黑客松大赛 — 作品星河长廊

> 信电学院 × 人工智能学院 · 18 个 AI 赋能校园的作品，汇成一片星河。

一个以「繁星 / 星座」为主题的**作品展示网站**，把首届 AI 黑客松的 18 支团队作品集中呈现：动画星空首屏、3D 立体轮播、按赛道筛选的星座网格，以及每个作品的详情灯箱（海报 + 队长 + 主张 + 简介 + 可选点赞评论）。

## ✨ 在线预览

- **GitHub Pages**：`https://1739467001-svg.github.io/zjsu-ai-hackathon/`
- 本地预览：`python3 -m http.server` 后浏览器打开

## 🎯 关于大赛

本届黑客松以**学院真实业务需求**为出发点，建立一套「从创意到可交付软件」的标准化流程——需求对齐、自主开发、材料提交与终评答辩，最终产出可演示、可复用、可继续优化的软件原型。

18 个作品覆盖 5 大赛道：

| 赛道 | 作品数 | 代表方向 |
|---|---|---|
| 智慧学务 | 5 | 人才简历筛选、学生事务、师生双选、新生选寝 |
| 评奖测评 | 3 | 奖学金评选 / 评定（双轨制综合测评、Flask+MCP+钉钉） |
| 空间预约 | 4 | SeatFlow、Smart Seat、座位 / 房间预约 |
| 科研实验 | 4 | 科研论文辅助管理、实验室 / 耗材管理 |
| 院务文化 | 2 | 线上院史馆、智慧党建 |

## 🧩 网站功能

- **首屏**：Canvas 实时星空（闪烁星点 + 流星 + 星轨）、发光主标题、数字滚动统计。
- **星河长廊**：3D Coverflow 立体轮播，自动播放 + 拖拽 + 方向键 + 圆点导航。
- **星座图谱**：按赛道筛选的作品网格，滚动逐颗「点亮」。
- **作品详情**：高清海报 + 赛道 + 标题 + 主张 + 简介 + 标签 + 队长，可选点赞 / 评论。
- **全终端响应式**：手机 / 平板 / 桌面自适应。

## 🛠 技术栈

纯静态前端，零构建：`HTML + CSS + 原生 JavaScript`。作品数据集中在 `assets/data.js`，可选的点赞 / 评论后端为 `server/api.py`（Python 标准库 + SQLite，零依赖）。

## 📁 目录结构

```
zjsu-ai-hackathon/
├── index.html              # 主页面
├── admin.html              # 后台管理（点赞/评论，需后端）
├── assets/
│   ├── style.css           # 样式
│   ├── app.js              # 交互（星空/轮播/网格/灯箱）
│   ├── data.js             # 18 个作品数据
│   └── works/              # 作品海报（+ thumbs/ 缩略图）
└── server/api.py           # 点赞/评论后端（可选）
```

## 📝 更新作品

所有作品信息都在 `assets/data.js`。改完后：改动 css/js/data 记得把 `index.html` / `admin.html` 里的 `?v=N` 同步 +1，换了海报把 `app.js` 顶部 `const VER` +1（避免浏览器缓存旧内容）。
